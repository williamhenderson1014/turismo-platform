ALCANCE DEL PROYECTO

El proyecto consiste en transformar el sitio web actual (HTML estático) en una plataforma dinámica y moderna con las siguientes funcionalidades:

SITIO PÚBLICO:
- Página de inicio con paquetes destacados y llamados a la acción
- Catálogo de paquetes turísticos con búsqueda y filtros por categoría
- Páginas de detalle de paquetes con itinerario completo, precios y disponibilidad
- Formulario de contacto y solicitud de cotizaciones
- Diseño responsive (móvil, tablet, escritorio)

PANEL DE ADMINISTRACIÓN:
- Dashboard con métricas clave (reservas, ingresos, cotizaciones pendientes)
- Gestión de Paquetes (CRUD completo con itinerarios, precios, imágenes)
- CRM de Clientes (historial de reservas, comunicación, segmentación)
- Gestión de Reservas (estados, pagos, confirmaciones)
- Sistema de Cotizaciones (seguimiento, respuestas, conversión)
- Reportes (ingresos por paquete, clientes top, análisis de conversión)
- Configuración (empresa, WhatsApp, pagos, notificaciones)

INTEGRACIONES:
- WhatsApp Business API (mensajes automáticos, notificaciones)
- Transbank Webpay (pagos con tarjeta de crédito/débito)
- Sistema de emails transaccionales

---

TECNOLOGÍAS PROPUESTAS

Backend: Laravel 11 con PHP 8.3
Frontend: Blade + Tailwind CSS + Alpine.js
Base de datos: MySQL 8.0
Pasarela de pago: Transbank Webpay Plus
Mensajería: WhatsApp Business API
Hosting: Servidor VPS con SSL
